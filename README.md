# E6792 Spring 2026: Lab4-YOLOv4-Tiny

This lab contains starter code for: Lab4-YOLOv4-Tiny. The formal lab assignment is distributed through GitHub Classroom. Populate the Lab4-YOLOv4-Tiny GitHub Classroom repo with the code in this repository. 

The GitHub Classroom repository will become (after students accept the GitHub classroom link invite) the student version of the lab for this class.
To complete the Lab4-YOLOv4-Tiny, students need to update their copy of this repository by putting their solution into the repository.

You are expected to complete the TODOs and discussion questions in the following files:

* Jupyter Notebook Part 1: `VMSetup.md`
* Jupyter Notebook Part 2: `DarknetSetup.ipynb`

* Jupyter Notebook Part 4: `ModelEvaluation.ipynb`
* Jupyter Notebook Part 5: `JITInferenceOptimization.ipynb`
* Jupyter Notebook Part 6: `WebcamInference.ipynb`

You are also expected to make the following files:

* Data folder: `darknet/data`

To ensure reception of full credit: 
* Upload versions of Jupyter Notebooks with appropriate cell outputs saved
